# 添加按钮颜色方案

## 设计要求
- 左侧栏选中状态：保留紫蓝色 (#6366F1)
- 添加按钮：需要更好看的颜色（非紫蓝色）

## 颜色方案选项

### 方案1：深灰黑（专业、克制）
```
背景：#1F2937（深灰）
文字：#FFFFFF（白色）
悬停：#111827（更深灰）
风格：专业、沉稳、企业级
```
**优点**：专业、永恒、不过时
**缺点**：可能略显沉重

### 方案2：翠绿色（现代、活力）
```
背景：#10B981（翠绿）
文字：#FFFFFF（白色）
悬停：#059669（深翠绿）
风格：现代、清新、有活力
```
**优点**：现代感强、视觉舒适
**缺点**：可能与"添加"语义不够强

### 方案3：天蓝色（清爽、友好）
```
背景：#3B82F6（天蓝）
文字：#FFFFFF（白色）
悬停：#2563EB（深天蓝）
风格：清爽、友好、易接近
```
**优点**：清爽、不突兀
**缺点**：可能与紫蓝色太接近

### 方案4：中性灰（极简、低调）
```
背景：#6B7280（中灰）
文字：#FFFFFF（白色）
悬停：#4B5563（深中灰）
风格：极简、低调、不抢眼
```
**优点**：极简、克制
**缺点**：可能不够突出

### 方案5：浅灰边框（最极简）
```
背景：#FFFFFF（白色）
文字：#1F2937（深灰）
边框：#E5E7EB（浅灰）
悬停：#F9FAFB（极浅灰背景）
风格：最极简、最轻盈
```
**优点**：最符合极简主义
**缺点**：可能不够醒目

## 推荐方案

### 🏆 推荐：深灰黑（方案1）
**理由**：
1. 与白色极简风格完美契合
2. 专业、永恒、不过时
3. 与紫蓝色选中状态形成对比
4. 符合企业级产品的气质

### 🥈 备选：翠绿色（方案2）
**理由**：
1. 现代感强
2. 与"添加"操作语义契合（绿色=新增）
3. 视觉舒适，不突兀

### 🥉 备选：浅灰边框（方案5）
**理由**：
1. 最符合极简主义
2. 最轻盈、最不抢眼
3. 适合追求极致简洁的设计

## 最终选择：深灰黑 (#1F2937)

采用深灰黑方案，理由：
- 专业、克制、永恒
- 与白色极简风格完美契合
- 不会过时的经典设计
- 与紫蓝色选中状态形成良好对比
