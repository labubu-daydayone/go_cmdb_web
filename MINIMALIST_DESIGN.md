# 极简主义设计方案

## 设计哲学

**Less is More** - 真正的极简主义不是简单地减少元素，而是：
- 去除一切不必要的装饰
- 使用最基本的色彩（黑白灰）
- 通过留白和层级创造美感
- 让内容成为焦点

## 色彩系统：无彩色（Achromatic）

### 主色调：黑色系
```
- 纯黑：#000000（强调、标题）
- 深灰：#18181B（主要文字、主按钮）
- 中灰：#52525B（次要文字）
- 浅灰：#A1A1AA（辅助文字、边框）
```

### 背景色：白色系
```
- 纯白：#FFFFFF（卡片、主背景）
- 极浅灰：#FAFAFA（页面背景）
- 浅灰：#F4F4F5（表格头部、次要背景）
```

### 状态色：克制使用
```
- 成功：#10B981（绿色，仅用于成功状态）
- 警告：#F59E0B（琥珀色，仅用于警告）
- 错误：#EF4444（红色，仅用于错误）
- 信息：#3B82F6（蓝色，仅用于信息提示）
```

## 按钮系统

### 1. 主按钮（Primary）
```
背景：#18181B（深灰黑）
文字：#FFFFFF（纯白）
悬停：#000000（纯黑）
圆角：6px
无边框、无阴影
```

### 2. 次要按钮（Secondary）
```
背景：#F4F4F5（浅灰）
文字：#18181B（深灰黑）
悬停：#E4E4E7（更浅灰）
圆角：6px
无边框、无阴影
```

### 3. 文字按钮（Text）
```
背景：透明
文字：#52525B（中灰）
悬停：#18181B（深灰黑）+ 下划线
无边框、无阴影
```

### 4. 危险按钮（Destructive）
```
背景：#EF4444（红色）
文字：#FFFFFF（纯白）
悬停：#DC2626（深红）
仅用于删除等危险操作
```

## 排版系统

### 字体
```
主字体：Inter（无衬线，现代感）
备选：-apple-system, SF Pro, Helvetica Neue
```

### 字重
```
- 标题：600（Semibold）
- 正文：400（Regular）
- 强调：500（Medium）
```

### 字号
```
- H1：28px（页面标题）
- H2：20px（区块标题）
- H3：16px（卡片标题）
- 正文：14px（主要内容）
- 小字：12px（辅助信息）
```

## 间距系统

采用 8px 基准网格：
```
- 4px：最小间距
- 8px：紧凑间距
- 12px：常规间距
- 16px：舒适间距
- 24px：区块间距
- 32px：大区块间距
- 48px：页面级间距
```

## 圆角系统

统一使用小圆角：
```
- 按钮：6px
- 卡片：8px
- 输入框：6px
- 对话框：12px
```

## 侧边栏设计

### 深色侧边栏（保持对比）
```
背景：#18181B（深灰黑）
文字：#FAFAFA（极浅灰）
图标：#A1A1AA（浅灰）
悬停：rgba(255, 255, 255, 0.1)
选中：#FFFFFF（纯白文字）
```

## 卡片设计

```
背景：#FFFFFF（纯白）
边框：1px solid #F4F4F5（极浅边框）
圆角：8px
阴影：无（或极浅 0 1px 2px rgba(0,0,0,0.05)）
悬停：轻微上移 -2px
```

## 表格设计

```
表头背景：#FAFAFA
表头文字：#18181B（深灰黑）
行边框：1px solid #F4F4F5
悬停行：#FAFAFA
```

## 输入框设计

```
背景：#FFFFFF
边框：2px solid #E4E4E7
聚焦边框：2px solid #18181B
圆角：6px
无阴影
```

## 极简主义原则

### 1. 去除装饰
- ❌ 渐变色
- ❌ 重阴影
- ❌ 多余边框
- ❌ 复杂动画
- ✅ 纯色块
- ✅ 极浅阴影
- ✅ 简洁线条
- ✅ 微交互

### 2. 层级清晰
- 通过字重区分重要性
- 通过字号区分层级
- 通过颜色深浅区分状态
- 通过留白分隔内容

### 3. 克制使用颜色
- 主要界面只用黑白灰
- 状态色仅用于状态指示
- 避免多种颜色混用
- 保持视觉统一

### 4. 留白为王
- 给内容足够呼吸空间
- 不要填满每个角落
- 用留白引导视线
- 简洁胜过复杂

## 参考案例

- **Stripe**：极简支付界面
- **Apple**：极简产品设计

## 实现优先级

1. ✅ 更新主题色彩为黑白灰
2. ✅ 移除所有渐变和重阴影
3. ✅ 统一按钮为无彩色系统
4. ✅ 优化排版和间距
5. ✅ 简化交互动画
6. ✅ 减少视觉噪音

## 预期效果

- **专业感**：黑白灰传达专业、可靠
- **现代感**：极简设计符合当代审美
- **易读性**：高对比度确保内容清晰
- **永恒性**：不会过时的经典设计
